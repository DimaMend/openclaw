# Ceramics Social Media Generator

Quickly generate Instagram posts for your ceramic work.

## Quick Start

```bash
# Interactive mode - prompts you for details
~/clawdbot/skills/ceramics-social/generate.sh

# Direct mode - skip prompts
~/clawdbot/skills/ceramics-social/generate.sh --style aesthetic --type new-work

# Show help
~/clawdbot/skills/ceramics-social/generate.sh --help
```

## Styles

| Style | Vibe | Best For |
|-------|------|----------|
| **aesthetic** | Minimal, evocative, visual | Finished pieces, portfolio shots |
| **technical** | Glaze details, process | Process shots, work-in-progress |
| **storytelling** | Personal, narrative | Artist journey, inspiration |
| **casual** | Friendly, behind-the-scenes | Studio vibes, casual sharing |

## Post Types

| Type | When to Use |
|------|-------------|
| **new-work** | Announcing a finished piece |
| **process** | Behind-the-scenes, WIP |
| **collection** | Showcase a series or theme |
| **story** | Narrative about a piece |
| **sale** | Available work announcement |

## Output

Each generated post includes:
- **Caption** - Ready to post
- **Hashtags** - Relevant tags for reach
- **Photo Suggestions** - Which shots to use
- **Engagement Booster** - Question or CTA

## Tips

- Copy the output directly to Instagram
- Use 3 photos max: overall + detail + lifestyle
- Post times: 9am, 12pm, or 7pm (local time) work best
- Reply to comments within 1 hour to boost algorithm
